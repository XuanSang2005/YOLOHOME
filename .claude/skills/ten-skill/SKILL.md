# UI Premium

## Rules
- Do not change backend logic.
- Do not break responsive layout.
- Improve spacing, hierarchy, and alignment first.
- Prefer subtle, premium polish over flashy effects.
- Keep the design clean and consistent.

## Workflow
1. Review the current UI first.
2. Identify the top visual problems.
3. Improve spacing, typography, and component consistency.
4. Add subtle polish only after structure is solid.
5. Re-check the final result for consistency.

## Output format
- Problems found
- Proposed improvements
- Changes made
